```md
# bootcamp-glb

Contains all the DSA questions from the Summer Bootcamp of GL Bajaj. <br>

## Java Array Questions <br>

1. **Count Consonants** (`countConst.java`) <br>
   Count the total number of consonants in a given string. <br>

2. **Find Largest / Smallest Element in an Array** (`larAndsml.java`) <br>
   Find the largest and smallest element present in an array. <br>

3. **Reverse Each Row of a Matrix** (`revMat.java`) <br>
   Reverse every row of a given matrix. <br>

4. **Sum of Boundary Elements of Matrix** (`boundSum.java`) <br>
   Find the sum of all boundary elements in a matrix. <br>

5. **Delete the Smallest Element from an Array** (`arrMindel.java`) <br>
   Delete the smallest element from a given array. <br>
```


## Java String Questions <br>

1. **Anagram Check** (`Anagram.java`) <br>
   Check whether two strings are anagrams or not. <br>

2. **Group Anagram** (`GpAnagram.java`) <br>
   Group or check multiple strings for anagram relationships. <br>

3. **Palindrome Check** (`PaliCheck.java`) <br>
   Check whether a string is a palindrome or not. <br>

4. **Password Generator** (`PasswordGen.java`) <br>
   Generate a password based on given conditions. <br>

5. **Remove Duplicate Characters** (`RemoveDuplicate.java`) <br>
   Remove duplicate characters from a string. <br>

6. **Remove Vowels** (`RemoveVowels.java`) <br>
   Remove all vowels from a string. <br>

7. **Replace String** (`Replacestr.java`) <br>
   Replace characters or words in a string. <br>
