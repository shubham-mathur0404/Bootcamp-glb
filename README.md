# bootcamp-glb

Contains all the DSA questions from the Summer Bootcamp of GL Bajaj.

## Java Array Questions

1. **Count Consonants** (`countConst.java`)  
   Count the total number of consonants in a given string.

2. **Find Largest / Smallest Element in an Array** (`larAndsml.java`)  
   Find the largest and smallest element present in an array.

3. **Reverse Each Row of a Matrix** (`revMat.java`)  
   Reverse every row of a given matrix.

4. **Sum of Boundary Elements of Matrix** (`boundSum.java`)  
   Find the sum of all boundary elements in a matrix.

5. **Delete the Smallest Element from an Array** (`arrMindel.java`)  
   Delete the smallest element from a given array.

---

## Java String Questions

1. **Anagram Check** (`Anagram.java`)  
   Check whether two strings are anagrams or not.

2. **Group Anagram** (`GpAnagram.java`)  
   Group or check multiple strings for anagram relationships.

3. **Palindrome Check** (`PaliCheck.java`)  
   Check whether a string is a palindrome or not.

4. **Password Generator** (`PasswordGen.java`)  
   Generate a password based on given conditions.

5. **Remove Duplicate Characters** (`RemoveDuplicate.java`)  
   Remove duplicate characters from a string.

6. **Remove Vowels** (`RemoveVowels.java`)  
   Remove all vowels from a string.

7. **Replace String** (`ReplaceStr.java`)  
   Replace characters or words in a string.
